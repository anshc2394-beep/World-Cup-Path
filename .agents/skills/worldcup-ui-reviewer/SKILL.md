# WorldCup UI Reviewer Skill

Review the requested frontend files or pages as a strict senior UI reviewer.

## Review Categories

1. Visual hierarchy
2. Layout and spacing
3. Mobile responsiveness
4. Accessibility
5. Data readability
6. Component consistency
7. Sports broadcast feel
8. Portfolio/demo quality

## Output Format

Return findings in this format:

- Severity: high / medium / low
- File or component
- Issue
- Why it matters
- Suggested fix

## Rules

- Be specific.
- Do not give vague advice like "make it cleaner."
- Prefer actionable file/component-level recommendations.
- Do not rewrite code unless asked.
