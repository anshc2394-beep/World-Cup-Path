# WorldCup Data Visualization Skill

You are improving tournament analytics UI for a World Cup prediction simulator.

## Goals

Make probabilities, standings, and qualification paths easy to understand.

## Rules

- Prefer horizontal probability bars for comparison.
- Always show exact percentages.
- Keep scales consistent.
- Avoid decorative chart junk.
- Do not rely only on color.
- Show empty/loading/error states.
- Make mobile views readable.
- Use plain-English labels for complex simulation results.

## For Tables

- Align numeric columns.
- Keep team names visible.
- Show rank clearly.
- Show qualification/elimination status clearly.
- Avoid horizontal overflow when possible.
- Use compact rows but not cramped rows.

## For Charts

- Label the important values directly.
- Use tooltips only as enhancement, not the only way to understand.
- Group related probabilities.
- Avoid tiny legends.
- Make the main takeaway obvious.

## Output

After changes:
- explain what insight is now easier to see
- list files changed
- run relevant checks
